# Qwen3.6-35B-A3B

Alibaba Qwen チームが 2026-04-16 にリリースしたオープンウェイト MoE モデル。  
35B パラメータを持ちながら推論時は 3B のみ活性化するため、32GB Unified Memory の Mac でも動作する。

## 概要

| 項目 | 値 |
|------|-----|
| アーキテクチャ | Sparse MoE |
| 総パラメータ | 35B |
| 推論時アクティブ | 3B |
| コンテキスト長 | 262K tokens |
| モデルサイズ | 24GB (Q4_K_M) |
| ライセンス | Apache 2.0 |
| 対応入力 | テキスト・画像 (マルチモーダル) |

## ベンチマーク

| ベンチマーク | スコア |
|-------------|--------|
| SWE-Bench Verified | 73.4% |
| MCPMark | 37.0 |
| MMLU-Pro | 85.3 |

## 強化された機能

- **Agentic Coding**: フロントエンドワークフローやリポジトリレベルの推論が向上
- **Thinking Preservation**: 過去メッセージの推論コンテキストを保持し、反復開発のオーバーヘッドを削減
- **Thinking モード**: `/think` `/nothink` でオン・オフ切り替え
